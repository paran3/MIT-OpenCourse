# TYPES OF KNOWLEDGE

declarative knowledge : 사실 그 자체를 의미한다.
imperative knowledge : "어떻게" 해당 사실이(결과) 발생하는지, 도췰되는지를 의미한다.

그렇다면 프로그래밍은 어떠한 특정한 사실을 도출해내기 위해 필요한 일련의 과정을 의미하는것인가?
